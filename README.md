# Nunu

Nunu é um projeto de pet virtual em desenvolvimento.

A ideia é criar um companheiro inteligente, capaz de interagir, guardar memórias e evoluir com o tempo. No momento, ele funciona pelo terminal, mas futuramente poderá ganhar interface gráfica, voz e até um corpo físico com Raspberry Pi e impressão 3D.

## Versão atual

Nunu v0.2

## Funcionalidades

- Sistema de humor
- Sistema de energia
- Sistema de fome
- Sistema de apego
- Respostas aleatórias
- Memória salva em arquivo JSON

## Comandos

- `oi`
- `status`
- `carinho`
- `comer`
- `brincar`
- `dormir`
- `sair`

python main.py
